Notes

The first version of this file uses fonts and is saved as an Inkscape SVG; therefore, it is a good start for editing (the text can be easily modified).

In the second version, I convert fonts to path, so that it will be displayed properly even on systems without the proper font installed (Bitstream Vera, which is free by the way), and I removed the Inkscape namespace; use it for display only.
